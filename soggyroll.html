Soggyroll

An anime streaming service

About

This is an app that is a clone of Crunchyroll, the anime streaming site. With this site you can create your own account, bookmark their favorite animes, search anime, as well as watch small bits of the anime series that you select.

  • Check it out!

Tech stack

This project uses a Ruby on Rails backend that connects to a PostgreSQL relational database, a React and Redux frontend, used to manage state and present a receptive frontend apprearence.

Features

Soggyroll features a few popular anime, as well as some personal favorites

  • An image carousel that will change every few seconds
  • A functional search bar that will be able to search for your favorite anime
  • You can watch episode clips from the respective anime
  • A favorite function, where you can save your favorite anime for later
  • Coming soon: comments, reviews, manga section, randomizer button, more anime and episodes
